“参数”这种变量有点不同，是为所谓系统变量，它不来自“赋予”而是传递

“参数”跟“参数栈”有何差别？

“参数”一般用于命令行或检查数据，每个参数前后会加引号，参数之间以空格分隔

“参数栈”则是把参数栈整体或局部引用过去

还是上代码

```
定义哦以下代码
	显示“命令是 echo ”‘参数【2】【3】’换行。
上代码。
哦1、2、3"、4。
定义啊以下代码
	显示‘参数0’‘参数’换行。
	哦‘参数栈【3】’。
上代码。
啊1、2、3、4、5、6、7。
```

得如下结果

```
命令是 echo "2" "3\""
啊"1" "2" "3" "4" "5" "6" "7"
命令是 echo "4" "5"
```
